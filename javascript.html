What is JavaScript?

JavaScript is a programming language that allows adding interactivity to web pages. It is one of the fundamental languages of web development along with HTML and CSS.

JavaScript was created by Brendan Eich in 1995. It runs in the web browser and allows dynamically modifying the web page content.

How JavaScript Works

JavaScript allows developers to perform various actions, such as:

  • Manipulating the DOM (Document Object Model).
  • Responding to user events, such as clicks or key presses.
  • Sending asynchronous HTTP requests with AJAX.

Basic JavaScript Example

Below is a simple example of JavaScript code that displays a message in the console:

<script>
    console.log("Hello, World!");
</script>
  • <script>: Indicates the start of a block of JavaScript code.
  • console.log(): Displays a message in the browser's console.
  • "Hello, World!": The text that will be displayed in the console.

JavaScript Variable Example

JavaScript allows you to store data using variables. Here's a simple example of how to declare and use a variable:

<script>
    let name = "John";
    console.log("Hello, " + name + "!");
</script>

This code will display "Hello, John!" in the console, using the variable name.

JavaScript Function Example

Functions in JavaScript allow you to encapsulate code and reuse it. Here's a simple example:

<script>
    function greetUser(username) {
        alert("Hello, " + username + "!");
    }
    greetUser("Alice");
</script>

In this example, the function greetUser is called with the argument "Alice", displaying an alert message "Hello, Alice!".

JavaScript Conditional Example

JavaScript conditionals allow you to execute different code based on certain conditions. Here's an example:

<script>
    let age = 18;
    if (age >= 18) {
        console.log("You are an adult.");
    } else {
        console.log("You are a minor.");
    }
</script>

This code checks if the age is 18 or older and outputs either "You are an adult" or "You are a minor" in the console.

JavaScript Loop Example

Loops allow you to repeat actions multiple times. Here's an example of a loop that prints numbers from 1 to 5 in the console:

<script>
    for (let i = 1; i <= 5; i++) {
        console.log(i);
    }
</script>

This loop will output the numbers 1, 2, 3, 4, and 5 in the browser's console.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Es uno de los lenguajes fundamentales del desarrollo web junto con HTML y CSS.

JavaScript fue creado por Brendan Eich en 1995. Se ejecuta en el navegador web y permite modificar el contenido de la página web de manera dinámica.

Cómo Funciona JavaScript

JavaScript permite a los desarrolladores realizar diversas acciones, como:

  • Manipular el DOM (Document Object Model).
  • Responder a eventos del usuario, como clics o teclas presionadas.
  • Enviar solicitudes HTTP asíncronas con AJAX.

Ejemplo de JavaScript Básico

A continuación, un ejemplo de un código JavaScript simple que muestra un mensaje en la consola:

<script>
    console.log("Hola, Mundo!");
</script>
  • <script>: Indica el comienzo de un bloque de código JavaScript.
  • console.log(): Muestra un mensaje en la consola del navegador.
  • "Hola, Mundo!": Es el texto que se mostrará en la consola.

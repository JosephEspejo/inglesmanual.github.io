What is HTML?

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It is used to structure and organize web content.

HTML was created by Tim Berners-Lee in 1991 and has become the foundational language for the web. HTML uses tags to instruct the browser on how to display content.

Basic HTML Structure

The basic structure of an HTML document is:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Page Title</title>
    </head>
    <body>
        <h1>This is a heading</h1>
        <p>This is a paragraph.</p>
    </body>
</html>
  • <!DOCTYPE html>: Declares the document type.
  • <html>: Marks the beginning of the HTML document.
  • <head>: Contains information about the document, such as the title and metadata.
  • <body>: Contains the visible content of the web page.
  • <h1>: Defines a heading on the page.
  • <p>: Defines a paragraph of text.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir el aspecto y la presentación de documentos HTML. Permite cambiar el diseño visual de los elementos de una página web.

CSS permite separar el contenido (HTML) de la presentación, lo que facilita la modificación del estilo sin afectar la estructura del documento. Se pueden cambiar colores, fuentes, tamaños, espaciado y mucho más.

Estructura Básica de CSS

La estructura básica de una regla CSS es la siguiente:

selector {
    propiedad: valor;
}

Por ejemplo, para cambiar el color de todos los párrafos a azul, usarías:

p {
    color: blue;
}

¿Qué son las transiciones en CSS?

Las transiciones en CSS permiten que los cambios de propiedades, como colores o tamaños, ocurran de manera gradual durante un período de tiempo.

Una transición básica se define usando la propiedad transition, especificando la propiedad a animar, la duración y el tipo de curva de animación (por ejemplo, ease, linear).

Ejemplo:

.custom-box {
    background-color: #f9f9f9;
    transition: background-color 0.5s ease;
}

.custom-box:hover {
    background-color: #007bff;
}

En este ejemplo, al pasar el cursor sobre el elemento con la clase custom-box, el color de fondo cambiará suavemente a lo largo de 0.5 segundos.

Tipos de CSS

Existen tres formas principales de aplicar CSS en una página web:

  • CSS en línea: Estilos aplicados directamente a los elementos mediante el atributo style. 
    <p style="color: red;">Texto en rojo</p>
  • CSS interno: Estilos dentro de la etiqueta <style> en la sección <head>. 
    <style>
    p { color: blue; }
</style>
  • CSS externo: Un archivo CSS separado que se vincula usando la etiqueta <link>. 
    <link rel="stylesheet" href="styles.css">

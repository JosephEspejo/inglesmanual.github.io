What is CSS?

CSS (Cascading Style Sheets) is a language used to describe the look and presentation of HTML documents. It allows you to change the visual design of web elements.

CSS allows you to separate content (HTML) from presentation, making it easier to modify the style without affecting the document's structure. You can change colors, fonts, sizes, spacing, and much more.

Basic CSS Structure

The basic structure of a CSS rule is as follows:

selector {
    property: value;
}

For example, to change the color of all paragraphs to blue, you would use:

p {
    color: blue;
}

What are CSS Transitions?

CSS transitions allow property changes, such as colors or sizes, to occur gradually over a specified period of time.

A basic transition is defined using the transition property, specifying the property to animate, the duration, and the animation curve type (e.g., ease, linear).

Example:

.custom-box {
    background-color: #f9f9f9;
    transition: background-color 0.5s ease;
}

.custom-box:hover {
    background-color: #007bff;
}

In this example, hovering over the element with the class custom-box will smoothly change the background color over 0.5 seconds.

Types of CSS

There are three main ways to apply CSS in a web page:

  • Inline CSS: Styles applied directly to elements using the style attribute. 
    <p style="color: red;">Text in red</p>
  • Internal CSS: Styles within the <style> tag in the <head> section. 
    <style>
    p { color: blue; }
</style>
  • External CSS: A separate CSS file linked using the <link> tag. 
    <link rel="stylesheet" href="styles.css">
